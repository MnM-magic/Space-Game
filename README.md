# Space Game

A 2D space shooter game developed in **Python** using the **Pygame** library during my **first year** of studying Computer Science at the **University of KwaZulu-Natal (UKZN)**. This was one of my earliest software development projects and was created to strengthen my understanding of programming fundamentals, object-oriented programming, and game development.

---

## Project Overview

The Space Game is a classic arcade-style shooter where players control a spaceship while avoiding or destroying enemies to achieve the highest possible score. Building this project introduced me to the principles of real-time game development, event-driven programming, collision detection, and interactive application design.

As one of my first major programming projects, it marked an important milestone in my software development journey and helped build the foundation for more advanced projects in Artificial Intelligence, Computer Vision, and Machine Learning.

---

## Features

- 🚀 Player-controlled spaceship
- 👾 Enemy spawning and movement
- 💥 Collision detection
- 🎯 Score tracking
- 🎮 Responsive keyboard controls
- 🖼️ Sprite-based graphics using Pygame
- 🔄 Real-time game loop
- 🎵 Interactive arcade-style gameplay

## Technologies Used

- Python
- Pygame
- Object-Oriented Programming (OOP)

## Educational Purpose

After developing this project during my **first year**, I later used it as part of presentations to **first-year Computer Science students** and during the **University of KwaZulu-Natal (UKZN) Open Day**.

The project demonstrated how the programming concepts introduced in first year—including variables, functions, loops, classes, object-oriented programming, and event handling—can be combined to create complete and interactive software applications.

My goal was to inspire prospective and first-year students by showing that the skills learned in introductory Computer Science courses can be applied to build creative, engaging, and real-world software projects.
